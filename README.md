# 🚀 PathNova - AI Career Exploration Platform

PathNova is an AI-powered platform designed to help students explore career paths, generate personalized roadmaps, and make informed decisions about their future.

---

## 📌 Features

* 🔐 Secure User Authentication (JWT-based)
* 🧠 AI-based Career Recommendations
* 🛣️ Personalized Career Roadmaps
* 🎨 Fully Responsive UI (Mobile, Tablet, Desktop)
* ⚡ Smooth Animations using GSAP
* 🔄 REST API Integration
* 💾 Database Management (MySQL)

---

## 🏗️ Tech Stack

### Backend:

* Java

* Spring Security
* REST APIs

### Frontend:

* Next.js
* React.js
* JavaScript
* HTML, CSS
* GSAP (Animations)

### Database:

* MySQL

### Tools & Others:

* Git & GitHub

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

git clone https://github.com/your-username/pathnova-career-exploration-platform.git

### 2️⃣ Backend Setup

cd backend
mvn clean install
mvn spring-boot:run

### 3️⃣ Frontend Setup

cd frontend
npm install
npm run dev
